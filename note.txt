git là phần mềm quản lý code (version control system)
github là nơi lưu trữ code online (iclound, driver...)
repository local trên laptop
repository remote trên github

git init: khởi tạo repo local
"git remote add origin"(câu lệnh bắt buộc có) https://github.com/GoldenMonkey799/bc67-demo-git.git , đây là liên kết repo local và repo remote

Đưa code lên github có 3 bước:
- git add -A
- git commit -m "nội dung commit"
- git push -u origin master (lần đầu tiên)/ git push (lần thứ 2 trở đi)
